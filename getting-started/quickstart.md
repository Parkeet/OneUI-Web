# Quickstart

Here what should look the html a page using OneUI CSS:
```
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <title>Hello, world!</title>
    <link rel="stylesheet" href="OneUI/oui.css">
  </head>
  <body>
    <div class="viewing">
        <!-- Viewing area -->
    </div>
    <div class="content">
        <!-- Interaction area -->
    </div>
  </body>
</html>

```
